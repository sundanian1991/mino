# Book Processor - Prompt 模板集合

---

## 核心 Prompt 模板

### 1. 章节结构分析

```
请分析这份文档的章节结构：

1. 共有几个章节？
2. 每章的标题是什么？
3. 每章大约从哪一行开始？
4. 是否有子章节？

请用表格形式输出：

| 章节 | 标题 | 起始行 | 结束行 | 预估页数 |
|------|------|--------|--------|----------|
| 01   |      |        |        |          |
```

---

### 2. 单章深度处理（完整版）

```
你是一位专业的学习笔记整理专家。请对这份内容进行深度学习处理：

## 处理要求

### 1. 段前批注
- 在每个重要段落前添加批注
- 使用图标：
  - 💡 核心洞察
  - 📌 关键点
  - 🎯 定义
  - ⚠️ 注意/易错
  - 📊 数据/图表
  - 🔑 关键概念
  - 🌟 重点推荐
- 批注简洁，不超过 50 字
- 批注要有点评感，不是简单复述

### 2. 表格整理
- 将列表式内容转为表格
- 对比类内容用对比表
- 分级类内容用分层表
- 表格要有表头说明

### 3. 图表可视化
- 用 ASCII 图展示逻辑关系
- 复杂流程用 mermaid（如适用）
- 框架图用文本框

### 4. 核心概念提炼
- 提取关键术语
- 给出清晰定义
- 标注重要程度（⭐⭐⭐）

### 5. 本章小结
- 3-5 个核心知识点
- 1-2 个关键洞察
- 用列表形式呈现

### 6. 思考题
- 3-5 道反思题
- 能引发深度思考
- 联系实际工作/生活

## 输出格式

- Markdown 格式
- 层级清晰（H1/H2/H3）
- 表格美观
- 中英文对照（如适用）

## 风格要求

- 批注有"人味"，像老师在旁边讲解
- 表格简洁，不花哨
- 思考题开放但有方向

请开始处理...
```

### 3. 单章处理（简化版）

```
请将这份内容整理为深度学习笔记：

要求：
1. 每段前加批注（💡📌🎯等图标）
2. 列表转表格
3. 添加本章小结
4. 设计 3 道思考题

保持批注简洁，表格清晰。
```

---

### 4. 批量处理

```
基于第一章的处理风格和格式，请处理第二/三/四章。

保持以下一致：
1. 批注风格（图标使用、长度、语气）
2. 表格格式（列名、样式）
3. 小结结构（知识点 + 洞察）
4. 思考题风格（开放式/封闭式）

这是第一章的样本：[附上第一章链接或内容]

请逐章处理，每章完成后我会检查。
```

---

### 5. 质量审查

```
请审查这份学习笔记的质量：

## 审查维度

1. **准确性**
   - 批注是否准确反映原意
   - 表格数据是否正确
   - 概念定义是否精准

2. **完整性**
   - 是否遗漏重要内容
   - 章节是否连贯
   - 思考题是否覆盖重点

3. **可读性**
   - 格式是否统一
   - 表格是否清晰
   - 批注是否简洁

4. **实用性**
   - 能否用于复习
   - 能否用于 AI 对话
   - 能否快速查找

## 输出

1. 列出发现的问题
2. 给出修改建议
3. 评分（1-10 分）
```

### 6. 内容修正

```
请修正以下问题：

1. [问题描述] → [期望修改]
2. [问题描述] → [期望修改]
3. [问题描述] → [期望修改]

请保持整体风格一致，只修改指定问题。
```

---

### 7. 章节对比

```
请对比第一章和第二章的处理结果：

1. 风格是否一致？
2. 格式是否统一？
3. 有无遗漏或偏差？
4. 需要调整的地方？

请给出具体修改建议。
```

---

### 8. 创建索引

```
请为这本书创建学习索引：

1. 章节目录（含页码）
2. 核心概念表（按字母/拼音排序）
3. 关键模型/框架列表
4. 思考题汇总

用 Markdown 表格形式输出。
```

---

### 9. 提取金句

```
请从这份内容中提取金句：

要求：
- 每章 3-5 句
- 有启发性/洞察力
- 标注出处（章节 + 页码）
- 可用于分享/发朋友圈

格式：
> 金句内容
> —— 章节名
```

---

### 10. 生成复习卡

```
请基于这份内容生成复习卡片：

每张卡片包含：
- 问题（正面）
- 答案（背面）
- 难度标记（⭐/⭐⭐/⭐⭐⭐）
- 关联章节

共生成 20 张卡片，覆盖所有核心知识点。
```

## 场景化 Prompt

### 处理教科书

```
这是教科书内容，请：
1. 强调定义和公式
2. 例题单独标注
3. 习题附带提示
4. 关键术语加粗
```

### 处理商业书籍

```
这是商业书籍，请：
1. 提取核心模型
2. 标注案例公司
3. 提炼可行动建议
4. 联系实际工作场景
```

### 处理技能教程

```
这是技能教程，请：
1. 步骤编号清晰
2. 注意事项标注⚠️
3. 常见错误单独列出
4. 添加练习建议
```

### 处理理论著作

```
这是理论著作，请：
1. 梳理理论脉络
2. 对比不同学派
3. 标注核心论点
4. 添加批判性思考题
```
