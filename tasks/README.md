---

## 作用

## 文档分类

### system/ - 系统文档
**生命周期**: 长期维护，手动更新
**示例**: SOP 流程、工具使用指南

### supplier/ - 供应商管理
**生命周期**: 项目周期 (90 天+)
**示例**: 联盟方案、管理思路、信息模板

### 供应商管理/ - 管理体系
**生命周期**: 长期维护
**示例**: 基础信息、管理体系、汇报框架

## 与记忆系统关系

```
tasks/ (任务库)          memory/ (记忆系统)
    │                        │
    │  任务完成 → 有洞察      │
    │  ───────────────────→   │ /observer 写入
    │                        │
    │  需要背景 → 读取        │
    │  ←───────────────────   │ /UPDATE_MEMORY 汇总
    │                        │
```

**区别**：
- `tasks/` = 可复用文档（主动查阅）
- `memory/` = 对话记录（按需加载）

---

## 维护规则

1. **新建任务文档** → 放在对应分类目录
2. **任务完成** → 有洞察则 `/observer` 记录
3. **文档过时** → 标记 `[已归档]` 或移动到 `_archive/`

---

## 相关命令

| `/observer` | 任务完成后记录洞察 |
